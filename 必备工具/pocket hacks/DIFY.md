## 使用

### 本地部署与启停

本地部署 dify 的启动与停止，在 `dify/docker` 目录下：

```bash
# start up dify -- modify configs in file 'docker-compose.yaml' or '.env'
docker compose up

# stop dify server and save the data volumn
docker-compose down
```
